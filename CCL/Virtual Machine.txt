Virtual machine
digital verison of physical computer
allows to run an operating system that completely behaves like a completely separate computer in an app window on desktop.

Virtual Machine Disk
It is a file format that describes containers for virtual hard drives to be used in virtual machines.

NAT Network
Network address translation
it is used to provide different private IP addresses to different users inside organisation's private network.

SCP
scp (secure copy) command in Linux system is used to copy file(s) between servers in a secure way. The SCP command or secure copy allows the secure transferring of files between the local host and the remote host or between two remote hosts.
